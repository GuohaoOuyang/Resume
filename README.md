## Installation

 `npm install`
 
 cd client 
 
 `npm install`
 
 cd ..

`npm start`

Runs the app in the production mode.
Open http://localhost:5000 to view it in the browser.

`npm run client`

run fontend only

`npm run server`

run backend only

`npm run dev`

run front and backend
